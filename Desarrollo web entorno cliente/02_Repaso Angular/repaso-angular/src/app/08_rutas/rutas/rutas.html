Elementos importantes de las rutas

Elemento Qué hace
app.routes.ts Define qué componente corresponde a cada ruta.
routerLink Sirve para navegar desde el HTML.
router-outlet Es el lugar donde Angular carga el componente de la ruta activa.
redirectTo Redirige a otra ruta automáticamente.

Ejemplo de ruta

En tu proyecto, una ruta como esta:

{{ '{' }}
  path: '03-binding',
  component: Binding
{{ '}' }}

Significa que cuando la URL sea /03-binding, Angular mostrará el componente Binding.
