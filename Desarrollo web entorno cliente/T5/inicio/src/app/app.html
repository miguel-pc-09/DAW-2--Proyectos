Binding

Para poder traer elementos que estan en lo logico y representarlos en la parte grafica, se utiliza el one-way binding. Por ejemplo, vamos a reenderizar una varibale: {{ nombre }} y voy a sacar un {{ nota }}

El event binding hace lo contrario. Lanza un evento y pasa datos desde lo grafico a lo logico. Para ello en el button, select, input asociamos un evento mediante (clik), ademas de definir el metodo en la parte logica
